MicOrb 使用教程

欢迎使用 MicOrb 👋！这是一款专为快捷控制麦克风设计的悬浮球工具，主打“一键闭麦、随时切换”，帮你解决自由麦场景下的尴尬困扰，操作简单，轻量无负担。以下是详细使用步骤，看完就能轻松上手～

一、前期准备（首次使用必看）

1. 启动 MicOrb，桌面会出现一个小巧的地球样式的悬浮球（白天就是开启麦克风，黑夜就是关闭咯）。
2. MicOrb不会关闭你的麦克风驱动，而是直接修改你麦克风的Mute状态，也就是直接给你麦克风静音啦。
3. 悬浮球会自动识别当前默认麦克风，进入就绪状态。

二、核心操作（最常用，一键掌握）

1. 一键切换麦克风状态（核心功能）
这是 MicOrb 最便捷的功能，无需打开软件主界面，直接按下快捷键即可：
- 右键悬浮球：可以打开小彩蛋，点击设置可以设置开关快捷键。
- 状态提示：
  - 麦克风开启：悬浮球显示白天的地球，此时你的声音可正常被收录。
  - 麦克风关闭：悬浮球显示黑夜的地球，此时麦克风静音，避免环境音、误说话尴尬。
2. 移动悬浮球（不遮挡操作）
悬浮球默认可自由拖动，避免遮挡桌面图标或其他软件界面：
- 拖拽悬浮球，拖动到桌面任意位置，松开即可固定。
- 悬浮球永远至于顶层，超级方便查看！

三、进阶设置（按需调整，更贴合你的习惯）

右键点击悬浮球，打开设置菜单，可调整以下功能（根据自身需求选择）：
1. 悬浮球样式：可调整大小。
2. 快捷键设置：自定义闭麦/开麦快捷键（如 Ctrl+Alt+M，或者是单个按键，也可以是鼠标侧键），无需右键点击悬浮球，键盘一键操作，更高效。

四、常见问题（遇到问题不用慌）

1. 切换麦克风状态没反应？
  - 确认当前麦克风是否正常（可在电脑设置中检查麦克风是否启用）。
2. 悬浮球消失了？
  - 可能是不小心拖到了桌面边缘，重启 MicOrb 即可重新显示。
3. 切换静音后，其他软件仍能收录声音？
  - 关闭其他可能控制麦克风的软件，避免冲突。
